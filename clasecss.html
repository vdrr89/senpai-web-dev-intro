CSS Introducción: selectores

En esta clase se explica como usar CSS dentro de la etiqueta head, aunque también se puede usar CSS en otro archivo aparte para luego "linkearlo" dentro de la etiqueta head.

Dentro del archivo html, dentro de la etiqueta head, creamos una etiqueta style, y dentro de eésta, elegimos un selector, por ejemplo p ó h1 y le asignamos un color, un tamaño de fuente.

En css se puede sobreescribir, osea, el navegdor va leyendo desde arriba hacia abajo, si primero le damos un color a p, y luego le damos otro color a p, finalemente se va a mostrar el último color

Los selectores pueden ser etiquetas, clases, id

En clase se hizo así:


    <!DOCTYPE html>
        <head>
            <title>Aquí va el title</title>
            <style>  <!-- Acá ponemos la etiqueta style y dentro el estilo -->
                body {
                    background-color: lightblue;
                }
                    
                h1 {
                    color: navy;
                    margin-left: 20px;
                }
            </style>
        </head>

        <body>
            <p>Aquí va todo el body</p>
        </body>
    </html>

También se puede hacer así:


    <!DOCTYPE html>
        <head>
            <link rel="stylesheet" href="aquí-la-ruta-y-nombre-del-archivo.css"> <!-- aquí se "linkea" el otro documento que contiene el css -->
            <title>Aquí va el title</title>
        </head>

        <body>
            <p>Aquí va todo el body</p>
        </body>
    </html>

Comentarios

Los comentarios se escriben con /* y cerrandose con */ de esta forma:

/* Comentario aquí */

Que puede servir de selector?

Pueden servir las etiquetas, como un p o un ul,
Pueden servir las clases, que se escriben con un punto delante, por ejemplo, para la clase titulos se escribiría .titulos
Pueden servir los id de los elementos, que se escriben con un # delante, por ejemplo, si el id se llama contenido1, el selector se escribe #contenido1 


p{
    color: red;
}

.titulos{
    font-size: 24px;
}

#contenido1{
    border: 1px solid coral;
}

Estos selectores funcionan, debido a que tienen su referencia en el html, por ejemplo, todos los p van a cumplir las propiedades que se escribieron en el CSS, todas las etiquetas que tengan la clase titulos y los id contenido1 van a cumplir las propiedades que se les asignó en el CSS.

Como asigno los selectores class e id en el html?

Las etiquetas como p ul img ya al escribirlas quedan designadas, las clases se colocan dentro de cada etiqueta en la siguiente forma: 


<p class="aquiLaClase">Aquí el contenido del párrafo</p>

Los identificadores (id) se colocan de forma silimar:


<div id="aquiElIdentificador"></div>

Colores Hexadecimales

Los colores hexadecimales tienen 6 digitos y se conforman así:

#aabf19

van desde el 0 hasta el 9 y continúa con la a hasta la f
se dividen de a 2 dígitos aa bb cc
siendo éstos r g b (red, green blue), ( #rrggbb )
cuando éstos grupos tienen la misma letra se pueden resumir a 3 dígitos, por ejemplo 

#aabbcc = #abc

referencia: el 000 blanco y la fff negro

Para agregarle transparecnia se le agregan 2 números al finalizar el color

#aabbcc80

RGB

RGB = Red Green Blue
Se usa para pantallas, se emite através de la luz
La combinación de estos colores forma el blanco, y la ausencia de los mismos el negro.

Se puede incluir RGB con transparencia, lo cual sería RGBA


    rgba(red, green, blue, alpha)
    rgba(255, 99, 71, 0.5)

Cascada, Herencia y Especificidad

Cascada

La cascada implica que el orden importa
El que se visualiza finalmente es la declaración que se encuentre por debajo,
Estos documentos leen como nosotros, de izquierda a derecha y de arriba hacia abajo.

Por ejemplo si a la etiqueta p le asignamos un color, luego escribimos más código, y luego a la misma etiqueta p le asignamos otro color, se mostrará el último color asignado.

Especificidad

Se refiere a que si son específicos, tiene más peso que los demás, sea que estén escritos por debajo o encima.

Orden dentro de la especificidad

  • id
  • class
  • p

Herencia

El elemento padre hereda sus propiedades a los elementos hijos,
No todas las propiedades son heredables.

Por ejemplo si asignamos un color a una ul, las li van a adquirir ese color,
Sin embargo, si le asignamos un color a una li, la ul no

las hereda.

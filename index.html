Colocando una imagen

Para indicar que queremos colocar una imagen usamos la etiqueta img, dentro de ella podemos poner el atributo alt, que sirve por ejemplo para usar en dispositivos de gente que no tiene buena audición o visión y necesita que el dispositivo lea lo que hay en la web, también es útil para que los motores de búsqueda puedan identificar cual es el contenido que hay en el sitio y que relevancia tiene.

Ésta etiqueta se usa de la siguiente forma:

<img src="https://aquí-la-url-de-la-imagen-o-su-ruta.jpg" alt="breve descripcion">

Ésta etiqueta no tiene cierre, solo se abre y listo.

Declaración HTML

Si bien luce como una etiqueta, en realidad es una declaración, ésta se coloca al inicio y se pone una de cierre al final del documento.
Sirve para decirle al navegador que clase de archivo está leyendo, para que pueda leerlo correctamente

Clases CSS

CSS Introducción: selectores

Dentro de un archivo html, dentro de la etiqueta head, hacemos una etiqueta style, y a la etiqueta (x ej) p le asignamos un color

En css se puede sobreescribir, osea, el navegdor va leyendo desde arriba hacia abajo, si primero le damos un color a p, y luego le damos otro color a p, finalemente se va a mostrar el último color

Se puede asignar atributos de estilo a etiquetas, así como a clases, o a id

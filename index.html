Clases HTML

Clase I: Introducción

Instalar Visual Studio Code, hacer un archivo html y usar las etiquetas: 

<h1> <p> <h3> <img> <ul> <li>

Éstas etiquetas significan:

  • p - párrafo
  • h1, h2, h3... - headings, encabezados, hay del 1 al 6
  • img - imagen
  • ul - lista desordenada
  • li - elemento dentro de la lista

La mayoría de las etiquetas (como por ejemplo, h1, h2, h..., p, ul, li, entre otras), tienen apertura y cierre 

<p>texto aquí</p>
<h1>texto aquí</h1>

Colocando una imagen

Para indicar que queremos colocar una imagen usamos la etiqueta img, dentro de ella podemos poner el atributo alt, que sirve por ejemplo para usar en dispositivos de gente que no tiene buena audición o visión y necesita que el dispositivo lea lo que hay en la web, también es útil para que los motores de búsqueda puedan identificar cual es el contenido que hay en el sitio y que relevancia tiene.

Ésta etiqueta se usa de la siguiente forma:

<img src="https://aquí-la-url-de-la-imagen-o-su-ruta.jpg" alt="breve descripcion">

Ésta etiqueta no tiene cierre, solo se abre y listo.

Estructuras

Declaración HTML

<!DOCTYPE html>
    Aquí en medio va todo el contenido del documento html
</html>

Si bien luce como una etiqueta, en realidad es una declaración, ésta se coloca al inicio y se pone una de cierre al final del documento.
Sirve para decirle al navegador que clase de archivo está leyendo, para que pueda leerlo correctamente

Según Wikipedia una declaración es:

Una declaración de tipo de documento o DOCTYPE (del inglés document type declaration) asocia un documento SGML o XML particular con una definición de tipo de documento.
Wikipedia

Semántica HTML

Los html tienen que explicarse a si mismo, tienen que ser descriptivos en sus etiquetas, esto es útil para quienes usan dispositivos con diversos tipos de accesibilidad, por ejemplo, personas visión reducida.

También es importante para los motores de búsqueda, por ejemplo, la etiqueta main determina que es lo más importante dentro de esa página, a diferencia de div que es una división no tan importante, los motores de búsqueda le van a dar más relevancia por ejemplo a la etiqueta main.

Ser claros al momento de escribir el código también nos ayuda a nosotros mismos a entendernos cuando vamos a abrir el documento, para saber que quisimos hacer, y en caso de trabajar en un equipo también ayuda a nuestros compañeros o colaboradores a entendernos entre todos.

Clases CSS

CSS Introducción: selectores

En esta clase se explica como usar CSS dentro de la etiqueta head, aunque también se puede usar CSS en otro archivo aparte para luego "linkearlo" dentro de la etiqueta head.

Dentro del archivo html, dentro de la etiqueta head, creamos una etiqueta style, y dentro de eésta, elegimos un selector, por ejemplo p ó h1 y le asignamos un color, un tamaño de fuente.

En css se puede sobreescribir, osea, el navegdor va leyendo desde arriba hacia abajo, si primero le damos un color a p, y luego le damos otro color a p, finalemente se va a mostrar el último color

Los selectores pueden ser etiquetas, clases, id

En clase se hizo así:


<!DOCTYPE html>
    <head>
        <title>Aquí va el title</title>
        <style>  <!-- Acá ponemos la etiqueta style y dentro el estilo -->
            body {
                background-color: lightblue;
            }
                
            h1 {
                color: navy;
                margin-left: 20px;
            }
        </style>
    </head>

    <body>
        <p>Aquí va todo el body</p>
    </body>
</html>

También se puede hacer así:


<!DOCTYPE html>
    <head>
        <link rel="stylesheet" href="aquí-la-ruta-y-nombre-del-archivo.css"> <!-- aquí se "linkea" el otro documento que contiene el css -->
        <title>Aquí va el title</title>
    </head>

    <body>
        <p>Aquí va todo el body</p>
    </body>
</html>

Variables

Una variable podríamos pensarla como una cajita, con un nombre y un contenido


    // Declarar una variable
    // let var const
    let nombre;

    // Asignar un valor a una variable
    nombre = "Veronica";

    //Declara y asignar un valor a una Variables
    let edad = 33;

    // Declarar una variable const
    const pi = 3.14;

    // No podemos reasignar un valor a una variable declarada con const
    pi = 3.15; // Error

    // Hacer y ver un arreglo
    let amigos = ["fulano", "mengano", "sultano"];
    //es un arreglo con 3 valores, para acceder a uno de estos valores...

    console.log(amigos[0]); // Fulano
    console.log(amigos[1]); // Mengano
    console.log(amigos[2]); // Sultano
    console.log(amigos[3]); // undefinded

    console.log(amigos.length); // 3

    // Hacer y ver un Objeto
    let persona = {nombre: Veronica, edad: 33, algo:true}

    console.log(persona.nombre) // Veronica
    console.log(persona.edad) // 33


Reglas de nombrado de variables

  • El nombre de la variable debe comenzar con una letra o un guión bajo (pero no un número)
  • El nombre de la variable solo puede contener caracteres alfanuméricos y guies bajos
  • Los nombres de las variables son sensibles a mayúsculas y miníscuilas
  • El nombre de la variable no puede ser una palabra reservada
